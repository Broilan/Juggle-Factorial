Overview

Jiggle Factorial is designed to mimic Brain HQ's "Juggle Factor," which serves to measure and improve multitasking ability, processing speed, working memory, and attention.

Game Play

Start the game by clicking the Start button. Follow the instructions and use the span and movement types to complete the levels. Use the settings to customize your game experience.

Span Types

Forward: The span will increase by one each time you get a correct answer.

Backwards: The span will decrease by one each time you get a correct answer.

Sequencing: Random numbers will be generated, and you'll return them in either ascending or descending consecutive order depending on whether you've selected forwards or backwards span.

Movement Types

Normal: Circles will move around the screen in random directions.

Rotation: Circles will rotate around a central point.

Game Settings

Flash Mode: Toggle flash mode to make circles flash intermittently.

Auto Progression: Automatically progress to the next level upon a correct sequence.

Show Answers: Display the correct sequence after an incorrect attempt.
